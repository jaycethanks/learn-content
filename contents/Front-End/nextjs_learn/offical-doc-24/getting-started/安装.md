## 安装

系统前置要求

-   > = Node.js 18.17

### 自动安装

官方推荐使用 [`create-next-app`](https://nextjs.org/docs/app/api-reference/create-next-app) 来创建一个 Next.js 应用，它会自动为你设定一切。

```bash
npx create-next-app@latest
```
