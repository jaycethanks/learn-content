# 1. OOP ?

OOP 表示着面向对象编程

**过程编程** 是关于编写对数据执行操作的过程或方法，而面向对象编程是关于 **创建既包含数据又包含方法的对象**。



面向对象编程相较于过程编程，有几个优点：

- OOP 执行快速，且简单
- OOP 提供了程序的清晰结构
- OOP 使得代码易于维护，修改和debug
- OOP 使得花少量代码和开发时间去重用代码成为可能
